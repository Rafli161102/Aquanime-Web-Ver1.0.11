Proyek Maskot: Aria Ayumi

Proyek Maskot adalah jantung visual komunitas AquaNime, di mana kami berkolaborasi untuk mengembangkan dan menghidupkan karakter maskot tercinta kami, Aria Ayumi. Aria bukan hanya simbol, tetapi juga sumber inspirasi untuk berbagai proyek kreatif lainnya.

Tujuan Proyek:

  • Mengembangkan identitas visual yang kuat untuk Aria Ayumi.
  • Menciptakan berbagai ilustrasi, pose, dan ekspresi Aria untuk keperluan branding dan promosi.
  • Mendorong anggota untuk berkreasi dengan Aria melalui fanart dan cosplay.
  • Menyediakan panduan karakter (character sheet) untuk konsistensi visual.

Aktivitas Utama:

  • Workshop Ilustrasi Karakter: Sesi belajar menggambar Aria dalam berbagai gaya.
  • Kompetisi Desain Kostum Aria: Mengajak anggota untuk merancang pakaian baru untuk Aria.
  • Pengembangan Character Sheet: Membuat referensi visual lengkap untuk Aria.
  • Kolaborasi dengan Cosplayer: Mendorong dan mendukung anggota yang ingin meng-cosplay Aria.

Ingin Berkontribusi?

Jika Anda seorang ilustrator, desainer karakter, atau memiliki passion dalam visual branding, proyek ini adalah tempat Anda! Kami selalu mencari talenta baru untuk membantu Aria berkembang.
